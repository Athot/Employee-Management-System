# Employee Management System (Role-Based Access)

A full-stack **Employee Management System** built with the MERN stack, designed to manage employees efficiently with **secure role-based access control**.

---

## Why this project?

This project demonstrates:

- Real-world business logic
- Role-based authentication & authorization
- Secure data handling using JWT
- Scalable full-stack architecture

---

## Tech Stack

**Frontend:**

- React.js
- Tailwind CSS
- React Router

**Backend:**

- Node.js
- Express.js

**Database:**

- MongoDB

**Authentication:**

- JSON Web Tokens (JWT)

---

## Roles & Access Control

| Role         | Permissions                        |
| ------------ | ---------------------------------- |
| **Admin**    | Add/remove employees, assign roles |
| **HR**       | View & edit employee details       |
| **Employee** | View own profile & data only       |

---

## Features

- Secure login with JWT authentication
- Role-based dashboards (Admin / HR / Employee)
- Admin can add & remove employees
- HR can update employee details
- Employees can view their own data only
- Attendance management system
- Leave management system
- Salary slip generation
- Protected routes & middleware

---

## Project Structure

/client → React frontend
